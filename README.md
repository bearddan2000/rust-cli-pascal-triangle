# rust-cli-pascal-triangle

## Description
Calculates binomial coefficient for a
given number, limit 10.

## Tech stack
- rust

## Docker stack
- ubuntu:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
